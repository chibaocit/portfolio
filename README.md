## 🎨 Customization

All design tokens live in the `:root` block at the top of `<style>`:

```css
:root {
  --accent:  #4af0b0;   /* primary green */
  --accent2: #7b61ff;   /* purple highlight */
  --accent3: #ff6b6b;   /* red accent */
  --bg:      #080a0f;   /* main background */
  --text:    #e8eaf0;   /* body text */
  /* ... */
}
```

To update content, search for your name, email, or section headings directly in `index.html` — everything is clearly commented.
